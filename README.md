## Installation

Clone this repo, npm install, init tinymce.

```bash
npm i
```

```bash
npm run postinstall-tiny-mce
```

## Usage

### Development server

```bash
npm start
```

You can view the development server at `localhost:4000`.

### Production build

```bash
npm run build
```

### Run docker image

```bash
docker-compose up
```

### Rebuild docker image

```bash
docker-compose up --build
```

### Stop docker image

```bash
docker-compose down
```

### Project run test

```bash
npm run test
```

### Project linting

```bash
npm run lint
```
